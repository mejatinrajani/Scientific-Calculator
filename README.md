# Scientific-Calculator (made in HTML, CSS and Javascript)
A responsive scientific calculator built with HTML, CSS, and JavaScript featuring arithmetic operations, trigonometric functions, logarithms, factorial, powers, constants (π, e), keyboard input, operation history, and a clean interactive UI — all running fully in the browser without external libraries.

## Features

### Basic Operations
- Addition, subtraction, multiplication, division
- Percentage calculation
- Square root
- Reciprocal (1/x)
- Sign change (+ / -)
- Backspace and clear options

### Scientific Functions
- Trigonometric functions (sin, cos, tan) — degree based
- Logarithm (log base 10)
- Natural log (ln)
- Exponential (eˣ)
- Power functions (x², x³, xʸ, 10ˣ)
- Factorial
- Mathematical constants (π, e)

### Smart Features
- Operation history display
- Operator highlighting
- Keyboard input support
- Error handling (e.g., division by zero)
- Precision formatting for decimal results
- Responsive UI design



##  Keyboard Controls

| Key | Action |
|---|---|
| 0–9 | Numbers |
| . | Decimal |
| + - * / | Operators |
| Enter or = | Calculate |
| Backspace | Delete last digit |
| Escape | Clear all |


## Built With

- HTML5
- CSS3
- Vanilla JavaScript (without any libraries or frameworks)
